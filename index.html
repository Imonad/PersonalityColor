情深意篤, 心靈契合
具備高頻率的正向互動且從不衝突，即使各自獨處也充分信賴，給與私人空間與時間，心靈高度契合。( < 0.1% )
相敬如賓, 相得益彰
關係穩定且互相尊重，衝突極少，雖不一定有強烈激情，但極具共識。( < 5% )
平凡克難, 相濡以沫
靠責任與長期習慣維繫，滿意度隨時間略微下滑。婚姻關係相對平穩但缺乏深層連結。( < 45% )
同床異夢, 各行其事
處於心理上的「分居」狀態。為了孩子或經濟原因維持婚姻，但缺乏情感交流。( < 30% )
貌合神離, 互不相讓
負向互動多於正向，關係搖搖欲墜，是離婚的高風險群。雙方各執己見缺乏溝通。( < 15% )
反目成仇, 勢不兩立
此階段通常已進入法律程序，即使同居也全為負面互動。充滿敵意與衝突。( < 5% )
